### How to run
```shell
# Clone Repository
$ git clone https://github.com/VinewZ/configs.git
```

```shell
# Install Dependencies
$ pnpm i

# Run Aplication
$ pnpm run dev
```
Go to http://localhost:5173/ to see the result.
